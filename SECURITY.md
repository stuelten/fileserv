# Security Policy

## Supported Versions

Currently, only the latest version of FileServ is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability in FileServ, please do NOT open an issue.
Instead, please report it by sending an email to the project maintainer.

We will acknowledge your email as fast as possible
and provide a timeline for the fix if the vulnerability is confirmed.
